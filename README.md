# 🍽️ Restaurant Management App

A web-based Restaurant Management System built using **React + Vite**.  
This application helps admins and customers manage restaurant operations efficiently.

## 🚀 Features

- Admin Dashboard
- Customer Dashboard
- Secure Login System
- Role-Based Authentication
- Update Restaurant Details
- Protected Routes
- Responsive UI

---

## 🛠️ Technologies Used

- React.js
- Vite
- React Router
- Context API
- CSS
- GitHub Pages (Deployment)

```bash
